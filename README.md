# Data Structures and Algorithms (DSA)

This repository contains Data Structures and Algorithms practice
programs implemented using Java. It focuses on building strong
problem-solving skills and understanding core DSA concepts.

## Topics Covered
- Arrays
- Strings
- Recursion
- Searching algorithms
- Sorting algorithms
- Basic time complexity concepts

## Folder Structure
- arrays/ – Array-based problems
- strings/ – String manipulation problems
- recursion/ – Recursive solutions
- searching/ – Linear and binary search
- sorting/ – Sorting algorithms

## Purpose
To develop a strong foundation in Data Structures and Algorithms
and prepare for coding interviews.
